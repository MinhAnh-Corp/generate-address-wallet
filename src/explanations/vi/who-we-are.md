# Cách Tôi Xây Dựng Ứng Dụng Web Này

Tôi đã sử dụng **Cursor** (một trình soạn thảo code được hỗ trợ bởi AI) để xây dựng ứng dụng web công cụ ví này, cùng với một chút kinh nghiệm trong phát triển web, bao gồm việc xây dựng explorer cho [STOChain](https://stochainscan.io/en).

## Tech Stack

- **React** với TypeScript
- **Vite** cho công cụ build
- **Ant Design** cho các component UI
- **Jotai** cho quản lý state
- **i18next** cho đa ngôn ngữ
- **Ethers.js** và **CosmJS** cho các thao tác ví blockchain

## Tính Năng

- Tạo cụm từ mnemonic BIP39
- Tạo cả ví EVM và Cosmos từ một mnemonic hoặc khóa riêng duy nhất
- Chuyển đổi địa chỉ Cosmos giữa các tiền tố khác nhau
- Chuyển đổi theme Dark/Light
- Hỗ trợ đa ngôn ngữ (Tiếng Anh/Tiếng Việt)
- Hoàn toàn client-side - không gửi dữ liệu đến server
